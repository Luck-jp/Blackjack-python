# 🃏 Blackjack Game (Python)

A command-line implementation of the classic Blackjack card game where the user plays against the computer.

---

## 🚀 Features

* User vs Computer gameplay
* Random card dealing system
* Blackjack detection
* Ace handling (11 → 1 adjustment)
* Game loop with replay option
* ASCII art interface

---

## 🛠️ Tech Used

* Python 3

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/Luck-jp/blackjack-python.git

2. Navigate to the folder:
   cd blackjack-python

3. Run the program:
   python main.py
